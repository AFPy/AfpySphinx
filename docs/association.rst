Charte AFPy
================

L'association possède une `charte`_ qui décrit les valeurs et un code de
conduite. Toute personne participant à un évènement de l'AFPy, qu'il soit
membre ou non, est tenu de se conformer à cette charte.

.. _charte: coc.html
